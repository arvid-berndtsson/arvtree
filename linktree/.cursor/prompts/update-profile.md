# Update Profile

Update the profile section of the Linktree page.

## Customizable Elements

In `src/App.tsx`, you can modify:

1. **Avatar Initials** - The text inside the avatar circle
2. **Name** - The `<h1>` heading
3. **Bio/Tagline** - The `<p>` subtitle text
4. **Contact Email** - The "Get in Touch" button mailto link

## Files to Update

- `src/App.tsx` - Main profile content
- `public/favicon.svg` - Update initials in the favicon
- `index.html` - Update meta tags (title, description, og:title, og:description)

## Remember

- Keep the bio/tagline short and punchy
- Update all meta tags for SEO and social sharing
- Ensure favicon initials match the profile
